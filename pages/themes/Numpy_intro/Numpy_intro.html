NumPy Intro

NumPy overview

numpy is a python package that adds support for:
Effective implementation of large, homogeneous multi-dimensional arrays
Great collection of high-level mathematical and logical functions to operate on these arrays
Built-in functions for linear algebra, Fourier transforms and random number generation
Written mainly in C language
Play nice with Matplotlib and SciPy
pandas is built on top of numpy

				# in activated virtual environment write:
				pip install numpy

N-Dimensional Arrays

The main object in numpy is the homogeneous N-Dimensional Array, called ndarray Object.
It is a sequence of elements (usually numbers), all of the same type, indexed by a tuple of positive integers for each dimension.
In numpy, dimensions are called axes
Number of axes defines the rank of the array

Example


			import numpy as np

			# create one-dimensional array
			arr1d = np.array([1, 2, 3])
			# create two-dimensional array
			arr2d = np.array([[1, 2, 3], [4, 5, 6]])

			print(f'arr1d: \n{arr1d}\n')
			print(f'arr2d: \n{arr2d}\n')

			print(f'arr1d.shape: {arr1d.shape}') # (rank 1, axes 0)
			print(f'arr2d.shape: {arr2d.shape}') # (rank 2, axes 0 and 1)
		

			arr1d:
			[1 2 3]

			arr2d:
			[[1 2 3]
			[4 5 6]]

			arr1d.shape: (3,)
			arr2d.shape: (2, 3)

Array with Mixed Data Types?

If you try to create a NumPy array with elements of different types, NumPy will upcast elements to a common data type that can accommodate all the values, usually converting everything to a type that can hold all the given data.

				import numpy as np

				mixed_array = np.array([1, 2.5, '3'])
				print(mixed_array.dtype)  # Output: <U32 (Unicode string)

NumPy Data Types

NumPy Data Types

NumPy provides a variety of data types, referred to as dtypes, which are more efficient than Python’s native data types.
These dtypes are crucial for memory efficiency and speed in numerical computations.
There are 5 basic numerical types representing booleans (bool), integers (int), unsigned integers (uint) floating point (float) and complex.
Those with numbers in their name indicate the bitsize of the type (i.e. how many bits are needed to represent a single value in memory)
Do not forget, that Numpy Arrays are homogeneous! I.e. can not store data from different data types.
Reference: Data types @numpy.org
