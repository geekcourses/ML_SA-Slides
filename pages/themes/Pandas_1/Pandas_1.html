pandas - Overview and Installation

Overview

pandas is an open source library providing high-performance, easy-to-use data structures and data analysis tools for the Python programming language
pandas is great for data analysis and modeling
The two primary data structures in pandas are Series and DataFrame
Pandas home page

Install with Anaconda

If you use Anaconda you don't need to install, as pandas is included.
To start using pandas just open a JupyterLab or VSCode from Anaconda.
Reference: Installation instructions

Install with Conda

You should install pandas in activated conda environment.

			# list your conda environments:
			$ conda info -e

			# activate conda (base) environment:
			$ conda activate base

			# install pandas on conda
			(base)$ conda install pandas

Install in Python's venv

You should install pandas in activated Python environment

			# create Python environment (if you do not have one):
			# note that executing next command will create virtual environment in current working directory:
			$ python3 -m venv .venv

			# activate Python (.venv) environment in Bash:
			$ source .venv/bin/activate

			# activate Python (.venv) environment in Command Prompt:
			$ .venv\Scripts\activate.bat

			# install pandas on conda
			(.venv)$ pip install pandas

Use Pandas

You can start using pandas in any Python file

				import pandas as pd

				df_users = pd.DataFrame({
					"name":['Maria', 'Petyr', 'Ivan'],
					"age":[23, 21, 34],
					"gender":['female', 'male', 'male']

or or in a JupyterNotebook

pandas - Series Object

Pandas Series Object

A Pandas Series is a one-dimensional array of indexed data, based on the NumPy ndarray.
But a Pandas Series object wraps both a sequence of values and a sequence of indices
Reference: Pandas Series

			import pandas as pd

			ds = pd.Series([1,2,3,4])
			print(ds)
		

			0    1
			1    2
			2    3
			3    4
			dtype: int64

Create a Series Object


			pandas.Series( data, index, dtype, copy)
data: array-like, dict, or scalar value
Contains data stored in Series
index: array-like or Index (1d)
Values must have the same length as data
dtype: any numpy datatype
If None, dtype will be inferred
copy: boolean, default False
Copy input data

Create Series with Explicit Indexing

The explicit index definition gives the Series object additional capabilities compared to numpy arrays
i.e. the index need not to be an integer, but can consist of values of any desired type

			ds = pd.Series([1,2,3,4], index=['a', 'b', 'c', 'd'])
			print(ds)
		

			a    1
			b    2
			c    3
			d    4
			dtype: int64

Create Series from dictionary

The dict keys will be used for index
If an index is passed, the values in data corresponding to the labels in the index will be pulled out.

			ds = pd.Series({
				"d":4,
				"a":1,
				"c":3,
				"b":2,
				"e":5
			})
			print(ds)
		

			a    1
			b    2
			c    3
			d    4
			e    5
			dtype: int64

Note

for Python version >= 3.6 and Pandas version >= 0.23:
the Series index will be ordered by the dict’s insertion order
for Python < 3.6 or Pandas < 0.23:
the Series index will be the lexically ordered list of dict keys

Series Indexing and Slicing

We can access series element by its position, using square bracket notation as in Python lists
We can access single series element by its index label in 2 ways:
square bracket notation (as in Python dictionaries)
dot notation (as in Python objects)

				# create Series Object:
				prices_ds = pd.Series(
							[1.5, 2, 2.5, 3],
							index=["apples", "oranges", "bananas", "strawberries"]
				)

				# access elements by position:


				# access element by label - dot notation:
				print( prices_ds.apples )

				# access elements by label - square brackets notation:
				print( prices_ds['apples'] )

Series slicing

the Series also supports array-style operations such as slicing
Note that, operations such as slicing will also slice the index.

			## slicing
			print(ds["a":"d"])
			#a    1
			#b    2
			#c    3
			#d    4
			#dtype: int64

Altering index in place

A Series’s index can be altered in place by assignment

			ds = pd.Series([1,2,3,4,5],index=["a","b","c","d","e"])

			ds.index = ["A","B","C","D","E"]
			print(ds)
			#A    1
			#B    2
			#C    3
			#D    4
			#E    5
			#dtype: int64
