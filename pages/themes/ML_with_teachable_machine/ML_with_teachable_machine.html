What is Teachable Machine?

What is Teachable Machine?

Google's Teachable Machine is a web-based tool that makes creating machine learning models fast, easy, and accessible to everyone.
It's simple and beginners-friendly.
The models you make with Teachable Machine are real TensorFlow.js models that work anywhere javascript runs, so they play nice with tools like Glitch, P5.js, Node.js & more.
Reference: teachablemachine.withgoogle.com

Simple Steps on building ML App with Teachable Machine

Gather data and group your examples into classes, or categories, that you want the computer to learn
