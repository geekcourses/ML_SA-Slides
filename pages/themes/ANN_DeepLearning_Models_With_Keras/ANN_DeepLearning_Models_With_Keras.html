Overview of Layer Types used in Deep Learning Modeling

Overview of Layer Types used in Deep Learning Modeling

Dense (Fully Connected) Layer

A Dense layer is a fully connected neural network layer where each neuron receives input from all neurons in the previous layer.
Used for most standard feed-forward networks. Ideal for feature extraction and classification tasks.

                from tensorflow.keras.layers import Dense

                # Adding a Dense layer
                dense_layer = Dense(64, activation='relu')

Convolutional Layer

Performs a convolution operation that captures spatial features from the input image or data.
Commonly used in image recognition tasks. Helps to reduce the number of parameters and computations.

                from tensorflow.keras.layers import Conv2D

                # Adding a Convolutional layer
                conv_layer = Conv2D(32, kernel_size=(3, 3), activation='relu')

Pooling Layer

Reduces the spatial dimensions (width and height) of the input, helping to reduce overfitting and improve efficiency.
Max pooling and Average pooling are the two common types of pooling layers.

                from tensorflow.keras.layers import MaxPooling2D

                # Adding a MaxPooling layer
                max_pooling_layer = MaxPooling2D(pool_size=(2, 2))

Dropout Layer

Randomly sets a fraction of the input units to 0 at each update during training time, which helps prevent overfitting.
Improves generalization by forcing the network to learn redundant representations.

                from tensorflow.keras.layers import Dropout

                # Adding a Dropout layer
                dropout_layer = Dropout(0.5)

Flatten Layer

Converts multi-dimensional input into a 1D vector.
Commonly used in transition from convolutional layers to dense layers.

                from tensorflow.keras.layers import Flatten

                # Adding a Flatten layer
                flatten_layer = Flatten()

Batch Normalization Layer

Normalizes the input of a layer by scaling and centering. Helps stabilize and speed up training.
Used between layers to maintain mean and variance of inputs.

                from tensorflow.keras.layers import BatchNormalization

                # Adding a Batch Normalization layer
                batch_norm_layer = BatchNormalization()

Activation Layer

Applies a non-linear function to the input. Popular activations include ReLU, Sigmoid, and Softmax.
Used to introduce non-linearity into the model, which helps in learning complex patterns.

                from tensorflow.keras.layers import Activation

                # Adding an Activation layer
                activation_layer = Activation('relu')
