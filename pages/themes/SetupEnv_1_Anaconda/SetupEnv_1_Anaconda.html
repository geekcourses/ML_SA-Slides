Anaconda Basics

Anaconda Basics

What is Anaconda?

Anaconda is a distribution of the Python and R programming languages for scientific computing (data science, machine learning applications, large-scale data processing, predictive analytics, etc.), that aims to simplify package management and deployment.
It contains thousands of open-source packages and libraries needed for data science and ML.
Anaconda Distribution or Anaconda Individual Edition, is a free and open-source product of Anaconda, Inc.
It is available for Windows, Linux, and macOS
Official Site: Anaconda Distribution

Why Anaconda for Data Science and Machine Learning?

Ease of Installation.
More than 8000 data science packages are available.
Anaconda installs the latest Python 2 or 3 version in an isolated and activated environment so any installed Python version doesn’t cause any issues for your projects.
It’s beginner-friendly! You don’t need any prior coding/programming knowledge that scares beginners away

Anaconda Features

Anaconda Repository
features over 8,000 open-source data science and machine learning packages, Anaconda-built and compiled for all major operating systems and architectures.
Conda
Conda is an open-source package and environment management system that runs on Windows, macOS, and Linux.
Anaconda Navigator
A desktop application that lets you easily manage integrated applications, packages, and environments without using the command line.
Cloud Environment Backup
You can connect Anaconda Navigator to the community portal: Anaconda Nucleus , to securely store your local environments in the cloud.

Miniconda

Miniconda is a free minimal installer for conda.
It is a small, bootstrap version of Anaconda that includes only conda, Python, the packages they depend on, and a small number of other useful packages, including pip, zlib and a few others.
While the Anaconda installation requires minimum 5 GB disk space to download and install, miniconda requires just 400 MB disk space to download and install.

Anaconda Installation

Anaconda Installation

System requirements

Operating system:
Windows 8 or newer, 64-bit macOS 10.13+, or Linux, including Ubuntu, RedHat, CentOS 7+, and others.
System architecture:
Windows- 64-bit x86, 32-bit x86;
MacOS- 64-bit x86 & M1;
Linux- 64-bit x86, 64-bit aarch64 (AWS Graviton2), 64-bit Power8/Power9, s390x (Linux on IBM Z & LinuxONE).
Space:
Minimum 5 GB disk space to download and install.
Reference: Installation @Anaconda Docs

Installation type

On Windows, macOS, and Linux, it is best to install Anaconda for the local user, which does not require administrator permissions and is the most robust type of installation.
