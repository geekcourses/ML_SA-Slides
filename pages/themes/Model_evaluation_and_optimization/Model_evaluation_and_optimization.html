Model evaluation metrics

Overview

Model evaluation metrics are used to assess how well a specific algorithm models the given data (i.e., how well the algorithm fits the data).
Metrics for Regression problems: Mean Absolute Error, Mean Squared Error, Root Mean Squared Error
Metrics for Classification problems: Classification accuracy, Confusion Matrix

Model evaluation procedures

Training and testing on the same data
the worst case! If you have enough data - never train and test on same data.
Train/test split
Split the dataset into two pieces, so that the model can be trained and tested on different data
Useful due to its speed, simplicity, and flexibility
K-fold cross-validation
Systematically create "K" train/test splits and average the results together
Runs "K" times slower than train/test split, but more accurate.
